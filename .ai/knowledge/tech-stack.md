# Tech Stack

Tài liệu này liệt kê các công nghệ cốt lõi được sử dụng trong dự án.

## 1. Frontend
- **Frameworks**: React (Next.js), Vue (Nuxt.js), Svelte (SvelteKit).
- **Language**: TypeScript.
- **Styling**: Tailwind CSS, CSS Modules, SCSS.
- **UI Components**: Radix UI, Shadcn UI, Headless UI, PrimeVue, Element Plus.
- **State Management**: Zustand, Pinia, Redux Toolkit, React Context.
- **Data Fetching**: TanStack Query (React/Vue Query).
- **Authentication**: Clerk, NextAuth.js, AuthJs, Firebase Auth.

## 2. Backend
- **Main Framework**: NestJS (Modular Architecture).
- **Other Frameworks**: Express, Fastify, Next.js (Server Actions/API Routes - Secondary).
- **Runtime**: Node.js.
- **Database**: PostgreSQL, MySQL, MongoDB.
- **ORM/Query Builder**: Prisma, Drizzle, TypeORM, Mongoose.
- **Authentication**: JWT, Clerk Integration.
- **API Style**: RESTful API, GraphQL, gRPC.

## 3. Infrastructure & DevOps
- **Deployment**: Vercel (Frontend), Docker (Backend).
- **CI/CD**: GitHub Actions.
- **Monitoring**: Sentry.

## 4. AI Tooling
- **IDE**: Cursor, WebStorm, VSCode.
- **Context Tool**: Context7 (MCP).

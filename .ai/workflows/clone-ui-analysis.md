Yêu cầu:

## Quy trình thực hiện
1. **Lập kế hoạch**: Xác định mục tiêu phân tích. **Trình bày kế hoạch phân tích và trình người dùng review.**
2. **Thực hiện phân tích**:
    - Phân tích layout tổng thể.
    - Liệt kê các section chính.
    - Xác định reusable components.
3. **Đề xuất**: 
    - Không viết code.
    - Chỉ mô tả cấu trúc UI và đề xuất folder structure trong Next.js App Router.

# 🧠 Architectural Decision Records (ADR)

## 📌 Khái niệm
Đây là nơi ghi lại các quyết định quan trọng về kiến trúc. Mỗi file đại diện cho một quyết định kèm theo:
- **Bối cảnh (Context)**: Tại sao cần đưa ra quyết định?
- **Giải pháp (Decision)**: Chọn hướng giải quyết nào?
- **Hệ quả (Consequences)**: Ưu/nhược điểm sau khi áp dụng.

# PR Description Prompt

Dùng prompt này để yêu cầu AI viết mô tả Pull Request chuyên nghiệp.

## Prompt mẫu
```text
Dựa trên những thay đổi tôi đã thực hiện trong code, hãy viết một mô tả Pull Request chi tiết bao gồm:
1. Tóm tắt các thay đổi chính (Summary).
2. Danh sách các file bị ảnh hưởng.
3. Hướng dẫn cách test các tính năng mới/sửa lỗi.
4. Ảnh hưởng đến hiệu năng hoặc bảo mật (nếu có).
Sử dụng ngôn ngữ: Tiếng Việt.
```
